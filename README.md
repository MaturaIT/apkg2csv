# Apkg to Csv

Converts .apkg Anki deck SRS file to .csv

## Requirements

- Python 3

## Installation

## Usage

copy .apkg file to project's root and run

```python
python src/run.py {filename}.apkg
```

it will output {filename}.apkg.csv

## Tests

```python
python -m unittest tests.TestFile
```
